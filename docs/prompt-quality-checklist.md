# Prompt Quality Checklist

Use this checklist before adding a prompt to a sales workflow.

## Input quality

- [ ] The prompt includes the company name.
- [ ] The prompt includes the target buyer.
- [ ] The prompt includes the industry.
- [ ] The prompt includes known context.
- [ ] The prompt states the desired output format.
- [ ] The prompt avoids confidential information.
- [ ] The prompt separates facts from assumptions.

## Output quality

- [ ] The output is structured.
- [ ] The output is concise.
- [ ] The output is relevant to the sales motion.
- [ ] The output avoids unsupported claims.
- [ ] The output includes useful next steps.
- [ ] The output is easy to review.
- [ ] The output can be reused by a sales rep.

## Sales usefulness

- [ ] The prompt helps save time.
- [ ] The prompt improves personalization.
- [ ] The prompt improves preparation.
- [ ] The prompt improves consistency.
- [ ] The prompt reduces manual work.
- [ ] The prompt supports better conversations.

## Risk control

- [ ] The prompt does not ask AI to invent facts.
- [ ] The prompt does not include sensitive customer data.
- [ ] The prompt does not request manipulative messaging.
- [ ] The prompt does not automate final decisions.
- [ ] The prompt makes room for human review.
